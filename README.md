# 💸 Smart Expense Tracker

A modern web application built using Django to efficiently track, manage, and visualize daily expenses with an interactive dashboard.

---

## 🌟 Overview

Smart Expense Tracker helps users monitor their spending habits by providing a clean interface to add, edit, delete, and analyze expenses. It also includes real-time analytics using charts for better financial insights.

---

## 🚀 Features

✔ Add, Edit, Delete Expenses  
✔ User Authentication (Login System)  
✔ Interactive Dashboard  
✔ Total Expense Calculation  
✔ Category-wise Expense Analysis  
✔ Data Visualization using Charts  
✔ Responsive & Modern UI  
✔ Secure User-based Data Handling  

---

## 🛠️ Tech Stack

| Technology | Usage |
|-----------|------|
| Django | Backend Framework |
| SQLite | Database |
| HTML, CSS, Bootstrap | Frontend |
| Chart.js | Data Visualization |
| Python | Core Logic |

---

## 📊 Key Highlights

- Built using **Django MVC architecture**
- Implemented **user-specific data filtering**
- Used **Django ORM (Aggregate functions)** for calculations
- Integrated **Chart.js for real-time analytics**
- Designed **modern glass UI with responsive layout**
